## 4. Key Features

### 🔐 회원가입 · 로그인
- 이메일 인증 기반 회원가입  
- 세션 기반 로그인 적용  
- 아이디/비밀번호 찾기 및 재설정
  
### 💬 커뮤니티
- **게시글**: 작성·수정·삭제 · 이미지/파일 첨부  
- **댓글/대댓글**: CRUD 기능  
- **검색·필터**: 키워드 필터링  

### 📝 관리자 기능
- **상품 등록**: 상품 등록, 삭제
- **회원 목록**: 회원 목록 출력
- **주문 목록**: 주문 목록 출력

### 🛠 상품 주문
- **상품 목록**: 카테고리 별 목록 출력
- **상품 상세정보**: 상품의 상세 정보를 출력
- **상품 구매**: 주문 정보를 입력 후 상품 구매
- **주문 페이지**: 주문 정보를 확인
